  1. Qus: What are the three componets of Web?

    Ans: The three components of web are HTML, CSS, and JavaScript.

  2. Qus: Define version control and give one example.

    Ans: Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later. An example of version control software is Github.

  3. Qus: What tag is used to create a paragraph in html? Give an example.

    Ans: The tage that is used to create a paragraph is <p>. For exaple: <p>This is a paragraph</p>.

  4. Qus: Identify the odd one from these tags: <span>, <p>, <a>, <i> and state the reason.

    Ans: The odd tag out is the <p> tag. The reason this is the odd tag out is beacause each of the other tags are used within the <p> tag.

  5. Qus: What are the tags required to create a navigation list in HTML?

    Ans: The tags required to make a navagtion list in HTML are <nav> and <a>.

  6. Qus: What element is used to create images? Name its attributes.

    Ans: The element used to create images is <img>.

  7. Qus: Identify the odd one from these tags: <table>, <tr>, <b>, <td> and state the reson.

    Ans: The odd tag out is the <b> because it bolds the text while the others are parts of a table.
